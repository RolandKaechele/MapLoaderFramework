# Required Unity Packages for MapLoaderFramework

This document lists the Unity packages and third-party assets required or recommended for the MapLoaderFramework project.


## Core Unity Packages

- **UnityEngine**
  - Provides core engine functionality, MonoBehaviour, serialization, Inspector attributes, etc.


## Third-Party Packages

These packages are external tools or assets that extend MapLoaderFramework's capabilities, such as supporting TMX map import, advanced rendering, or enhanced input handling.


### 1. SuperTiled2Unity

- **Source:** [SuperTiled2Unity](https://github.com/Seanba/SuperTiled2Unity)
- **Purpose:** Imports Tiled (.tmx) maps as Unity prefabs for use in MapLoaderFramework.
- **Integration Steps:**
  1. Download from GitHub or Unity Asset Store.
  2. Import the package into your Unity project.
  3. Place your Tiled map files in the appropriate folder (e.g., `Assets/InternalMaps`).


## Optional Packages

- **TextMeshPro**
  - For advanced text rendering in UI.
- **Unity Input System**
  - For modern input handling.

For further details, see the individual package documentation and integration guides.
